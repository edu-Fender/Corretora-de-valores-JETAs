EXTRAIR O ARQUIVO "CorretoraDesktop.zip".

CRIAR BANCO DE DADOS MS SQL SERVER DE ACORDO COM O SCRIPT 'Corretora.sql'.

----------------------------------------------------------------------------------------------------------------------------------------

ABRIR O ARQUIVO "Corretora.exe.config".

ATUALIZAR A STRING DE CONEXÃO '<connectionStrings>...</connectionStrings>', 
ENTRAR COM AS INFORMAÇÕES DO BANCO DE DADOS 'Server=NOMEDOSERVIDOR;Database=NOMEDOBANCODEDADOS;Trusted_Connection=True',
ENTRAR COM AUTENTICAÇÃO DO BANCO DE DADOS (SE FOR NECESSÁRIO).

NÃO ALTERAR O NOME DA STRING DE CONEXÃO 'name="string"' DE MANEIRA ALGUMA. 

CONSULTAR "https://www.connectionstrings.com/" PARA MAIS INFORMAÇÕES DE COMO FORMATAR A STRING DE CONEXÃO.
